# twitter-history

Track your Twitter followers and followings in a GitHub repository.

## Local Setup

Clone the repo and install dependencies.

```console
git clone https://github.com/invisiblefunnel/twitter-history.git
cd twitter-history
bundle install
cp sample.env .env
```
